# 🛒 E-Commerce API Testing Project

## 📌 Overview
This project demonstrates API testing using Postman on a real-world E-commerce API.

## 🚀 Features
- Authentication (Login API)
- CRUD Operations (Cart APIs)
- Postman Assertions
- Negative Testing
- Environment Variables

## 🧪 Test Coverage
- Status Code Validation
- Response Time Validation
- Response Body Validation
- Edge Cases (Empty, Null, SQL Injection)
- Unauthorized Access Testing

## 🛠 Tools Used
- Postman
- JavaScript (Tests tab)
- FakeStore API

## 📂 Project Structure
ecommerce-api-testing/
│
├── postman_collection.json
├── test_cases/
├── environment/
└── screenshots/

## ⚙️ Setup Instructions

1. Import `postman_collection.json` into Postman  
2. Import environment file  
3. Set `base_url = https://fakestoreapi.com`  
4. Run Login API first to generate token  
5. Execute all requests
